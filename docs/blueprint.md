# **App Name**: PathFinder Ai

## Core Features:

- Resume Parsing: Upload a resume (PDF or DOCX) and extract skills/experience using the OpenAI API.
- AI Chat Mentor: Chat with an AI mentor (OpenAI GPT) for personalized career advice, skill-gap analysis, and learning recommendations. Supports multilingual interaction (English + Indian regional languages). The LLM uses tool use, so it can reason about if and when it should incorporate the data from your resume.
- Career Path Dashboard: Visualize top 3 career paths based on the user profile and skill match. Each path includes required skills, growth stats, and a roadmap progress bar. The roadmap is displayed as a visual timeline with milestones.
- Job Market Trends: Display graphs and charts for skill demand, salaries, and job openings using a job trend API (placeholder for MVP).
- Mock Interview Simulator: Simulate AI mock interviews with GPT, ask role-specific questions, score answers, and give improvement suggestions.
- User Authentication: Allow login via Google / Email to store basic user profiles (name, goals, skills, past sessions).

## Style Guidelines:

- Primary color: A cool blue (#42A5F5) evokes trust and professionalism, aligning with career guidance.
- Background color: A very light blue (#F0F8FF) provides a clean, modern backdrop.
- Accent color: A vibrant orange (#FF7043) is used for CTAs and highlights to draw attention.
- Body text: 'Inter' (sans-serif) for a modern, neutral, and readable style.
- Headline font: 'Space Grotesk' (sans-serif) is used for headlines. If longer text is anticipated, 'Inter' is used for the body
- Modern, clean icons representing various career paths, skills, and job market trends.
- Glassmorphic dashboard layout with clear separation of modules, smooth transitions, and responsive design.
- Subtle animations on content loading and user interactions to improve user experience.
